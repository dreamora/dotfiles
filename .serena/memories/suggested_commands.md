# Suggested Commands

## Development
- `shellcheck <script.sh>` - Lint shell scripts
- `stylua nvim/` - Format Lua files
- `eslint <file.js>` - Lint JavaScript

## Installation
- `./install.sh` - Full system setup (run from Terminal.app, not iTerm)
- `./install_packages.sh <profile>` - Profile-based package install (combined/private/business)
- `brew bundle` - Install from Brewfile

## Git (custom aliases in .gitconfig)
- Conventional commits: `git feat`, `git fix`, `git docs`, `git chore`, `git refactor`
- With scope: `git feat -s scope "message"`
