# GSD Project Knowledge

Use this project-local skill as a stable knowledge surface for GSD planning and execution.

## What This Skill Contains

- Mirrored knowledge cards generated from `docs/solutions/` compound docs.
- Category index for fast lookup.

## How To Use During Planning/Execution

1. Read `references/index.md`.
2. Identify categories relevant to the current phase goal.
3. Read matching cards under `references/compound/<category>/`.
4. Apply reuse guidance and known pitfalls to plan/execution decisions.
